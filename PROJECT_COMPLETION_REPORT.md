## 核心配置说明

### AI服务选择

**使用DeepSeek（推荐国内用户）：**
```env
AI_SERVICE_PROVIDER=deepseek
DEEPSEEK_API_KEY=sk-your-deepseek-key
DEEPSEEK_MODEL=deepseek-chat
```

**使用OpenRouter：**
```env
AI_SERVICE_PROVIDER=openrouter
OPENROUTER_API_KEY=sk-or-v1-your-key
OPENROUTER_MODEL=openai/gpt-4o
```
