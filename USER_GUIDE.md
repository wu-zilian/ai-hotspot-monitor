# AI热点监控工具 - 用户指南

## 快速开始

### 环境要求
- Node.js 18+
- MongoDB 4.4+
- npm 或 yarn

### 启动项目

#### Windows用户
```bash
# 双击运行
start.bat
```

#### Linux/Mac用户
```bash
# 添加执行权限并运行
chmod +x start.sh
./start.sh
```

### 首次使用

1. **访问应用**
   - 前端地址: http://localhost:3000
   - 后端地址: http://localhost:5001

2. **注册账户**
   - 点击"注册"创建新账户
   - 填写用户名、邮箱和密码

3. **配置AI服务**
   - 进入"系统设置"
   - 选择AI服务商（推荐使用DeepSeek）
   - 填写API密钥

4. **添加关键词**
   - 进入"关键词管理"
   - 点击"添加关键词"
   - 输入要监控的关键词

5. **开始监控**
   - 点击"立即扫描"按钮
   - 系统将自动采集并分析热点

## 功能说明

### 关键词管理
- **添加关键词**：输入关键词名称和描述
- **编辑关键词**：修改关键词信息
- **删除关键词**：移除不需要的关键词
- **搜索过滤**：快速查找关键词

### 热点监控
- **自动扫描**：系统按设定间隔自动扫描
- **手动扫描**：点击"立即扫描"触发
- **实时更新**：WebSocket推送最新热点
- **进度跟踪**：查看扫描和分析进度

### 通知设置
- **邮件通知**：配置SMTP服务
- **Web推送**：应用内实时通知
- **通知历史**：查看历史通知记录

### 系统设置
- **扫描配置**：设置扫描间隔和数量
- **置信度阈值**：设置AI分析的最低要求
- **账户管理**：修改个人信息和密码

## 常见问题

### Q: 如何获取AI API密钥？
**A:**
- **DeepSeek**: 访问 https://platform.deepseek.com/
- **OpenRouter**: 访问 https://openrouter.ai/

### Q: 如何配置邮件通知？
**A:**
1. 进入"系统设置" → "通知设置"
2. 填写邮箱地址和授权码
3. 点击"测试邮件通知"验证配置

### Q: 扫描失败怎么办？
**A:**
1. 检查网络连接
2. 确认AI服务配置正确
3. 查看浏览器控制台错误信息
4. 检查后端服务是否正常运行

### Q: 如何提高扫描效率？
**A:**
1. 合理设置扫描间隔（建议30分钟以上）
2. 优化关键词数量和质量
3. 提高置信度阈值减少无效数据
4. 使用稳定的AI服务

### Q: 数据存储在哪里？
**A:**
- MongoDB数据库: localhost:27017
- 数据库名: ai-hotspot-monitor
- 可通过MongoDB管理界面查看: http://localhost:8081
